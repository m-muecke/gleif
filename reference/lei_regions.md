# Fetch LEI regions

Fetches the list of regions from the GLEIF API.

## Usage

``` r
lei_regions()
```

## Value

A [`data.frame()`](https://rdrr.io/r/base/data.frame.html) with columns:

- **code**: The region code

- **language**: The language of the region name

- **name**: The region name

## Examples

``` r
if (FALSE) { # \dontrun{
lei_regions()
} # }
```
